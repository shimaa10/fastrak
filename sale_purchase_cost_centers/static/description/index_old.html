Sale, Purchase, Invoice, Employee and Expense Cost Centres

Basically cost centres app will allow to keep track expenses in Odoo, where manager are responible for accounting expense for all sorts, based on this tracking and monitering of expense cost accounting company improve budget allocation in proper manner to avoid unneccesary costing and expense from all department of company , This module helps user to add cost center in sale, purchase, invoice, employee and expense in Odoo accounting module

Cost Center on Sales.

User can add cost center on sales order and this cost center will apply on sales order line, user can change cost center as per need.

Cost Center on Purchase .

User can add cost center to purchase order and this cost center will apply on purchase order, user can change cost center of order and order line if needed.

Cost Center on Invoice, Employee, & Expense.

User can add cost center to customer invoice and vendor bill, also in employee and expense, user can change cost center of order and order line if needed.

Description

Configurations

In order to create and add cost center user have enable "Analytic Accounting" under Technical Settings.



Cost Center Menu

Under Invoicing > Configuration > Cost Center menu user can create cost center.


Cost Center on Sales

On sales order user can select cost center and this selected cost center will apply on sale order lines, user can also change cost center on order and order line.



Cost center on order line will change based on cost center on order.



Cost Center on Purchase

On purchase order user can select cost center and this selected cost center will apply on purchase order lines, user can also change cost center on order and order line.


Cost Center on Customer Invoices

User can add cost center on customer invoice and invoice line.



Cost Center on Vendor Bill

User can add cost center on vendor bill and and bill lines.


Cost Center on Employee

User can add and apply cost center for employee.



Cost Center on Expense

User can see employee cost center in expense.



Cost Center on Expense Report

User can see expense cost center also apply in expense report and expense report order line.
